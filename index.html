Accès non autorisé

Cette page contient des données personnelles et privées protégées par chiffrement.

Vous tentez d'accéder à des fichiers appartenant à une personne privée.
Tout accès non autorisé est illégal.

Ce message est affiché à quiconque visite le repo GitHub sans le bon mot de passe.

Structure du repo GitHub
1
Crée un repo GitHub (public ou privé — privé recommandé pour double sécurité)
2
Crée un dossier /photos et upload tous tes fichiers .enc
3
Upload ce fichier HTML (vault.html → index.html) à la racine du repo
4
Active GitHub Pages dans les settings du repo (Source: main branch)
5
Ton vault est accessible sur https://[username].github.io/[repo]
Workflow quotidien
→
Ajouter une photo : Onglet "Chiffrer" → chiffre → upload le .enc dans /photos du repo
→
Consulter : Télécharge les .enc depuis GitHub → Onglet "Visualiser" → entre le mot de passe
→
Sécurité : Même si quelqu'un accède au repo, les .enc sont illisibles sans ton mot de passe
Sécurité technique
🔑
Algorithme : AES-256-GCM — standard militaire
🧂
Dérivation du mot de passe : PBKDF2 avec 200 000 itérations + sel aléatoire
🌐
100% local — aucune donnée ne quitte ton navigateur pendant le chiffrement
📂
Chaque fichier a son propre sel et IV uniques — impossible à attaquer en masse
